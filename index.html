April 18, 2026

Context Menus & AI-Free Search (v0.1.4)

  • Full Context Menu: Fixed a major bug where right-clicking didn't work. Implemented a native context menu with Back/Forward/Reload, "Open Link in New Tab," and standard text actions.
  • ✨ Raw Copy: Added a special "Raw Copy" feature to the right-click menu that copies selected text as pure plain text, stripping all links and formatting.
  • AI-Free Search (Beta): Implemented a "Block Google AI Overview" setting that seamlessly prevents AI summaries from appearing. Added focus-protection to ensure the "-noai" tag stays hidden while typing.
  • Security Hardening: Resolved 11 vulnerabilities (including 9 high-severity issues) by upgrading electron-builder to v26.8.1 and refreshing dependency locks.
  • added credits: added credits to the leef home page, including a custom SVG Canadian flag and detailed project attribution.
  • Branding & UI Fixes: Enlarged internal page logos and matched the "Credits" button style with the rest of the UI.
April 17, 2026

v0.1.3 — Launch Release

  • Settings Persistence: All user preferences (ad blocker, HTTPS-only, search engine, zoom, tracking level, etc.) now persist across restarts via localStorage and are sent to the main process on startup.
  • Home Page Search Bar: Added a search/URL input directly on the home page so you don't have to rely on the top address bar.
  • Dynamic Leef Hub: Hub tiles can now be added and removed by the user. Click the × on any tile to remove it, or click "+ Add More" to add a custom site. Your layout persists across sessions.
  • News Feed "I Don't Care": Each news card now has a dismiss button that replaces it with the next available headline from the RSS feed.
  • News Feed Refresh: Added a refresh button next to the Leef News heading to fetch the latest headlines on demand.
  • Toast Notification System: Replaced all native alert() dialogs with a sleek, animated toast notification system used across bookmarks, settings, data clearing, and site warnings.
  • YouTube Warning Toast: A one-time toast warns users about potential performance impacts above 1440p and known fullscreen border limitations.
  • Video Fullscreen Support: Entering fullscreen on videos now hides the browser chrome. The main process handles OS-level fullscreen via webContents events.
  • Background Activity Limits: Background tabs now pause all video and audio elements (not just mute) and resume playback when you return to that tab.
  • Security Hardening: Added webSecurity flag, RSS content sanitization (XSS prevention), URL link validation, empty input guards, webRequest handler stacking fix, and a 512KB RSS data cap.
  • Performance: Settings save is debounced (500ms), news images use lazy loading, dead code removed, and ToastManager initialization ordered correctly.
  • Window Controls Color Match: Title bar overlay color now matches the CSS top bar exactly.
  • Color-Shift Fix: Forced sRGB/OpenGL rendering to fix grayed-out colors on static screens (Windows HDR issue).
  • GitHub Security Policy: Added SECURITY.md with vulnerability reporting process and response timeline.
April 16, 2026

Bug Fixes & Full Settings Implementation

  • Window Drag Fixed: Resolved an issue where users couldn't drag the browser window from the top bar. Interactive elements (buttons, tabs, address bar) are now correctly excluded from the drag region.
  • Viewport Highlight Fixed: Fixed a bug where clicking and dragging caused the entire browser UI to highlight in blue. Text selection is now disabled across the browser chrome for a native-app feel.
  • Default Profile Avatar: Replaced the external placeholder profile image with a locally stored default avatar asset that works fully offline.
  • Ad Blocker Implemented: The Ad & Tracker Blocking toggle now actively intercepts and cancels network requests to known ad/tracker domains using Electron's webRequest API.
  • HTTPS-Only Mode: The browser now automatically upgrades http:// requests to https:// at the network layer when this setting is enabled.
  • Tracking Protection Levels: Standard and Strict modes now use separate block lists. Strict mode blocks 20+ additional analytics, fingerprinting, and retargeting domains.
  • Notification Permissions: The master notification toggle now properly controls whether websites can request notification permissions via Electron's permission handler API.
  • Downloads — Ask Where To Save: Enabling "Ask where to save" now triggers a native Save dialog before any file begins downloading.
  • Background Activity Limits: Inactive tabs now have audio muted automatically when background limits are enabled, reducing resource usage.
  • Startup Behavior: All three startup modes (New Tab, Continue Where Left Off, Open Homepage) are now fully functional. "Continue" mode persists tab URLs across sessions.
  • Color-Shift Fix: Resolved an issue on Windows/HDR where colors would appear "grayed out" or washed out when the screen was static. Forced sRGB rendering and disabled direct video overlays.

The Leef Story

Leef was designed in 2022 as a proof of concept and was originally just a very basic web page, not a standalone browser. Around 2023, you would go there instead of google.com. It was popular in a small group of about 30 people, but eventually, it was abandoned.

Development on the full browser started in July 2025, and it was released into public alpha in April 2026. It was made to be a fast and AI-free alternative to Google.

Every browser company is adding AI slop to their browser. I got tired of it, so I made the opposite: a no-AI browser.
